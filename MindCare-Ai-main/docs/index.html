🧠 MindCare AI

AI-Powered Mental Health Support Platform

📖 About MindCare AI

MindCare AI is a production-style, Python-first, self-hosted emotional support and awareness platform. It provides AI-powered tools for mental health awareness and support while maintaining strict safety guidelines.

⚠️ Important Disclaimer: MindCare AI is not a medical professional and does not replace therapy, diagnosis, or professional mental-health care.

✨ Key Features

🤖 AI-Based Testing

Text analysis for emotional assessment with risk detection and personalized advice

💬 Intelligent Chat

Multi-turn, context-aware conversations with empathy-first responses

📚 Mental Health Dictionary

Comprehensive resource for understanding mental health concepts and phobias

📊 Personal Dashboard

Track your mental health trends with visual analytics and insights

🛠️ Technology Stack

Python 3.10+ Flask MySQL 8+ HTML5 CSS3 JavaScript Chart.js

🚀 Quick Start

1. Setup Environment

# Create virtual environment python -m venv venv # Activate (Windows) venv\Scripts\activate # Install dependencies pip install -r requirements.txt

2. Database Setup

# Create MySQL database and user mysql -u root -p < db/schema.sql

3. Run Application

python run.py

Visit: http://localhost:5000

🔧 Configuration

Configure your application using environment variables:

  • MYSQL_HOST - Database host (default: localhost)
  • MYSQL_USER - Database username
  • MYSQL_PASSWORD - Database password
  • MYSQL_DATABASE - Database name
  • SECRET_KEY - Flask session secret

📱 Available Pages

  • Home: Welcome page with feature overview
  • Test: AI-based mental health assessments
  • Chat: Interactive AI chat support
  • Dictionary: Mental health education resources
  • Dashboard: Personal analytics and trends
  • Contact: Get in touch with support

🔗 API Endpoints

  • POST /api/v1/predict - Text analysis
  • POST /api/v1/chat/message - Chat interaction
  • GET /api/v1/mcq - Assessment questions
  • POST /api/v1/contact - Contact form
